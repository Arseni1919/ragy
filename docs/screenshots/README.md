# Screenshots Directory

This directory is for storing screenshots referenced in the main README.md file.

## Required Screenshots

Please add the following screenshots to make the README complete:

1. **banner.png** - Project banner/logo image (hero section)
2. **cli-interface.png** - Full CLI interface showing logo and menu
3. **create-index.png** - Screenshot of index creation process
4. **xray-timeline.png** - Xray visualization showing timeline plot
5. **stats.png** - Database statistics display
6. **mcp-integration.png** - Claude Desktop using MCP tools

## Image Guidelines

- **Format**: PNG or JPG
- **Resolution**: At least 1200px wide for banners, 800px for screenshots
- **Quality**: High quality, readable text
- **Dark/Light**: Consider both dark and light terminal themes

## Usage

Images are referenced in README.md with relative paths:
```markdown
![Description](docs/screenshots/filename.png)
```

Make sure filenames match exactly with the references in README.md.
